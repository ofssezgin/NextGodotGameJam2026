# NextGodotGameJam2026
This game made for Next Godot Game Jam
23.05.2026 - 26.05.2026

Idle, Clicker, Pixel Art

Geleceğini İnşa Et, kulübeyle başlayan şehir kurma maceranızı üniversitelerle tamamlayın. 
Satın alabileceğiniz bonuslarla daha fazla gelecek puanı kazanın.

Geleceğini İnşa Et, binalara tıklayar gelecek puanı kazandığınız ve bu gelecek puanlarıyla 
şehrinizi inşa ettiğiniz bir oyun. 
Pixel art görseller, oyun tasarımı ve kodlama bana aittir.
